## Run Locally

Clone the project

```bash
  git clone https://github.com/yubint/commerce/
```

Go to the project directory

```bash
  cd commerce
```

Install requirements.txt

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python manage.py runserver
```
